# AI Usage Report — Assignment 3

## 1️⃣ Tools Used & Use Cases

During this assignment, I used the following AI tool:

**ChatGPT**

### How I Used It

* Identifying and fixing coding errors in HTML, CSS, and JavaScript.
* Assisting in implementing interactive features such as:
  - Navigation active section highlighting during scrolling.
  - Dark / Light theme toggle with state persistence using localStorage.
  - Expand / collapse behavior for project details.
* Supporting the integration of an external API to dynamically fetch and display inspirational quotes.
* Assisting in implementing form validation with multiple conditions (e.g., empty fields, email format, message length).
* Providing suggestions to improve layout, styling, responsiveness, and user experience.
* Reviewing my portfolio structure to ensure it meets the assignment rubric requirements.
* Supporting documentation writing such as README and reports.

AI was used as a support tool to guide development and problem-solving, not to replace my own work.
